Hospital Management System – SQL Database Project

This repository contains a complete Hospital Management System Database built using MySQL. The project demonstrates database design, table relationships, SQL queries, joins, aggregate functions, window functions, and data manipulation operations used in real-world hospital management systems.

Features
Patient management
Doctor management
Appointment scheduling
Medical records tracking
Billing and payment management
Department management
Doctor–Department relationship mapping
Database Concepts Covered
Database creation and table design
Primary & Foreign Keys
One-to-Many and Many-to-Many relationships
INSERT, UPDATE, DELETE operations
Filtering and sorting queries
Aggregate functions (SUM, AVG, COUNT, MAX, MIN)
Joins (INNER, LEFT, RIGHT)
Subqueries
Window functions (RANK, COUNT OVER)
Date and string functions
CASE statements for categorization
Tables Included
Patients
Doctors
Appointments
MedicalRecords
Billing
Departments
Doctor_Department
Sample Functionalities
Register patients and doctors
Schedule and manage appointments
Store diagnoses and prescriptions
Generate billing records
Analyze doctor performance and hospital revenue
Track patient visit history
Technologies Used
MySQL
SQL Queries
Relational Database Management System (RDBMS)
Purpose of the Project
Database management learning
Academic mini-projects
Backend database reference for hospital applications
